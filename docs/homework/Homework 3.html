import numpy as np
def beta(a,w):
    return 1.122 - 0.231*a/w + 10.55*(a/w)**2 - 21.71*(a/w)**3 + 30.82*(a/w)**4
def KI(a,w,s):
    return s*np.sqrt(np.pi*a)*beta(a,w)

For an infinitely wide, center-cracked panel we use (2.1)

KI=σπa−−√KI=σπa

In plane strain, the plastic stress intensity factor, KIeKIe is given by

KIe=σπ(a+rp)−−−−−−−−√KIe=σπ(a+rp)
where (in plane strain)
rp=16π(KIeσYS)2rp=16π(KIeσYS)2

Substituting rprp into KIeKIe gives

KIe=σπ(a+16π(KIeσYS)2)−−−−−−−−−−−−−−−−−⎷KIe=σπ(a+16π(KIeσYS)2)

We square both sides to find

K2Ie=σ2π(a+16π(KIeσYS)2)KIe2=σ2π(a+16π(KIeσYS)2)
Multiplying out we get
K2Ie=σ2πa+σ26(KIeσYS)2KIe2=σ2πa+σ26(KIeσYS)2
We can subtract the second term from both sides
K2Ie−σ26(KIeσYS)2=σ2πaKIe2−σ26(KIeσYS)2=σ2πa
And simplify
K2Ie(1−σ26σ2YS)=σ2πaKIe2(1−σ26σYS2)=σ2πa

We can now divide both sides by (1−σ26σ2YS)(1−σ26σYS2) to find

K2Ie=σ2πa1−σ26σ2YSKIe2=σ2πa1−σ26σYS2
We take the square root of both sides
KIe=σπa−−√1−σ26σ2YS−−−−−−−√KIe=σπa1−σ26σYS2
We can now replace σπa−−√σπa with KIKI
KIe=KI1−σ26σ2YS−−−−−−−√KIe=KI1−σ26σYS2
And divide both sides by KIKI
KIeKI=11−σ26σ2YS−−−−−−−√KIeKI=11−σ26σYS2

Now we are ready to generate our plot. Fracture mechanics is only valid when σ<σYSσ<σYS, so we consider 0<σ<σYS0<σ<σYS for our plot.
